## How to run the project

### API

- `cd server`
- `python -m venv venv`
- `./venv/Scripts/activate`
- `pip install -r requirements.txt`
- `python main.py`

### Website

- `cd web`
- `npm install` or `yarn install`
- `npm run dev` or `yarn dev`
